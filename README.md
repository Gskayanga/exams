# Exams Portal

This is a web-based portal designed for managing and accessing exam materials. 

## Features
* **Secure Login**: Includes a login interface with Google Authentication support.
* **Responsive Design**: Styled to work on both mobile and desktop browsers.
* **Apolitical Section**: Dedicated content area for apolitical materials.

## How to Access
You can view the live project here: [https://gskayanga.github.io/exams/](https://gskayanga.github.io/exams/)

## Project Structure
* `index.html`: The main landing page.
* `login.html`: The user authentication page.
* `js/auth.js`: Logic handling user login and security.
* `docs/`: Folder containing project documentation and assets.
